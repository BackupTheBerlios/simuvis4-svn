Help on Help
============
Like most non-commercial projects, SimuVis' documentation is not complete. Volunteers
are very welcome! When available, the help function (*'F1'*) gives you documentation
in your preferred language. If the there's no translated version available, it will
automatically deliver the english version.

This documentation is written in reStructuredText_, other formats (like HTML) are generated
from this source. If you read this via SimuVis' Help function, it's presented by an internal
webserver and can be viewed with any webbrowser.
The plugin documentation will be inserted at runtime.
