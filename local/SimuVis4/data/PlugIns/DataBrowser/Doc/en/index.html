PlugIn: DataBrowser

This plugin provides a common area for browsing data organised in lists and trees. It appears as a dockwidget in the main window which can be toggled with 'CTRL-B'. The different types of data are organised in a toolbox, every toolbox may contain a browser. A filesystem browser and a simple netCDF3 browser is provided for convenience.

This plugin may be used by other plugins (like DataStorageBrowser).

This plugin is far from being complete!

Configuration

In a configfile you may use the following items:

[databrowser]

# bool: show browser dockwidget at startup
show = no

# bool: show netcdf3 browser
netcdf3_browser = yes
